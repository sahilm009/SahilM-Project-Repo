Features

Real-time Ethereum transaction analysis via BigQuery
Retention analysis showing boost for early swappers
A/B testing simulation with volume lift results
Product strategy recommendations (swap nudges, gas alerts, etc.)
Interactive dashboard with charts and metrics

Quick Start
1. Install Dependencies
pip install streamlit plotly pandas numpy google-cloud-bigquery seaborn matplotlib
2. Download Files
Save these 2 files in the same folder:

ethereum_dashboard.py (Streamlit app)
enhanced_ethereum_analytics.py (Analytics engine)

3. Run Dashboard
bashstreamlit run ethereum_dashboard.py

The dashboard will open in your browser.
